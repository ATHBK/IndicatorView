# IndicatorView
A library for indicator viewpager
How to use:
- in file buid.gradle of project, add:
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
  
- in file buid.gradle of module, add:
dependencies {
	  compile 'com.github.ATHBK:IndicatorView:v1.0'
}
